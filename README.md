# Next.js scaffolding with JEST, Tailwind, and Storybook

Bootstraped using [Create Next App](https://github.com/vercel/next.js/tree/canary/packages/create-next-app#readme)!

## Stacks

- Next.js
- Tailwind
- JEST
- Storybook

## Scripts

### Next scripts
```bash
next dev
next build
next start
```

### Run Storybook

```bash
npm run storybook
```

### Run Tests

```bash
npm test
```
